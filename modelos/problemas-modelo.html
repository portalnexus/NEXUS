Conjunto de Problemas 1: Triângulos Retângulos

Problema 1.1: O Caminho do Parque

Imagine um parque retangular com 80 metros de comprimento e 60 metros de largura. Um novo caminho reto é construído conectando dois cantos opostos do parque.

Qual o comprimento deste novo caminho?

Dica: O caminho, juntamente com dois lados do parque, forma um triângulo retângulo.

Solução (Clique para revelar)

Mostrar/Ocultar Solução

Seja 'a' o comprimento do parque (80m) e 'b' a largura do parque (60m). O caminho 'c' é a hipotenusa.

Usando o teorema de Pitágoras: \(a^2 + b^2 = c^2\)

\(80^2 + 60^2 = c^2\)

\(6400 + 3600 = c^2\)

\(10000 = c^2\)

\(c = \sqrt{10000}\)

\(c = 100\) metros

Resposta: O novo caminho tem 100 metros de comprimento.

Conjunto de Problemas 2: Noções Básicas de Álgebra e Probabilidade

Problema 2.1: O Número Misterioso

Se você pegar um certo número, multiplicá-lo por 4 e depois adicionar 7, o resultado é 35. Qual é o número?

Dica: Monte uma equação linear para representar o problema. Deixe 'x' ser o número misterioso.

Solução (Clique para revelar)

Mostrar/Ocultar Solução

Seja 'x' o número misterioso.

O problema pode ser escrito como a equação: \[4x + 7 = 35\]

Passo 1: Subtraia 7 de ambos os lados para isolar o termo com x.

\(4x + 7 - 7 = 35 - 7\)

\(4x = 28\)

Passo 2: Divida por 4 para resolver para x.

\(\frac{4x}{4} = \frac{28}{4}\)

\(x = 7\)

Resposta: O número misterioso é 7.

Problema 2.2: Jarra de Bolinhas de Gude

Uma jarra contém 5 bolinhas de gude vermelhas, 3 azuis e 2 verdes. Se você pegar uma bolinha da jarra aleatoriamente, qual a probabilidade de pegar uma bolinha azul?

Dica: Primeiro, encontre o número total de bolinhas na jarra.

Solução (Clique para revelar)

Mostrar/Ocultar Solução

Passo 1: Encontre o número total de bolinhas de gude (o tamanho do espaço amostral).

Total de bolinhas = 5 (vermelhas) + 3 (azuis) + 2 (verdes) = 10 bolinhas.

Passo 2: Identifique o número de resultados favoráveis (pegar uma bolinha azul).

Número de bolinhas azuis = 3.

Passo 3: Calcule a probabilidade.

\[P(\text{blue}) = \frac{\text{NNúmero de bolinhas azuis}}{\text{Total de bolinhas}}\]

\(P(\text{blue}) = \frac{3}{10}\)

Resposta: A probabilidade de pegar uma bolinha azul é \(\frac{3}{10}\) ou 0,3.
